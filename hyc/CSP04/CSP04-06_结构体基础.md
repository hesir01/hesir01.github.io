# CSP04-06 结构体基础

## 一、课程简介（5分钟）

### 🎯 课程目标

1. 理解结构体的作用：将多种类型数据组合成一个自定义类型
2. 掌握结构体的定义、变量声明、成员访问（`.` 运算符）
3. 理解结构体的内存空间计算（内存对齐原则）
4. 掌握结构体数组的定义与遍历，以及配合 sort 进行排序

### 📚 核心知识点

- 结构体定义：`struct 名称 { 成员; };`
- 成员访问：`变量名.成员名`
- 结构体初始化：`{值1, 值2, ...}`
- 结构体空间计算（内存对齐）：`sizeof`
- 结构体数组：`类型名 数组名[n];`
- 结构体数组的 sort 排序：配合自定义 `cmp` 函数

## 三、新知讲解（45分钟）

### 1. 新知导入 🎬

> 用生活中的"档案袋"来理解结构体：
>
> 每个学生有一个档案袋，里面装了：学号（int）、姓名（string）、成绩（int）、身高（double）。
>
> 普通变量就是把学号、姓名、成绩分别放在不同的抽屉里；
> **结构体**就是把这些信息都装进一个档案袋，通过档案袋（变量名）+ 隔层（`.`）找到里面的具体信息！
>
> C++ 结构体：
> ```cpp
> struct Student {      // 定义"档案袋"的格式
>     int id;           // 学号
>     string name;      // 姓名
>     int score;        // 成绩
> };
>
> Student stu;          // 创建一个档案袋
> stu.id = 1;           // 在里面填学号
> stu.name = "小明";    // 填姓名
> ```

### 2. 知识点讲解

#### 2.1 结构体的定义

**语法格式：**

```cpp
struct 结构体名 {
    类型1 成员名1;
    类型2 成员名2;
    // ...
};   // 注意：结尾有分号！
```

**示例：**

```cpp
struct Student {
    int id;          // 学号
    string name;     // 姓名
    int score;       // 分数
    double height;   // 身高
};
```

> ⚠️ 注意：结构体定义结尾的 `}` 后面必须有分号 `;`，这是 C++ 语法规定！

---

#### 2.2 结构体变量的定义与使用

**定义结构体变量：**

```cpp
Student stu1;             // 定义一个 Student 类型的变量
struct Student stu2;      // 也可以加 struct 关键字（C 风格）
```

**访问成员（`.` 运算符）：**

```cpp
stu1.id = 1001;
stu1.name = "小明";
stu1.score = 95;

cout << stu1.name << ": " << stu1.score << endl;
```

**结构体变量赋值（整体赋值）：**

```cpp
Student stu2 = stu1;   // 整体复制，两个变量完全独立
stu2.name = "小红";    // 修改 stu2 不影响 stu1
```

---

#### 2.3 结构体的初始化

**方式一：按成员顺序初始化（最常用）**

```cpp
Student stu = {1001, "小明", 95, 170.5};
```

**方式二：指定成员名初始化（C++11 后支持）**

```cpp
Student stu = {.id = 1001, .name = "小明", .score = 95};
```

**方式三：先定义，再逐个赋值**

```cpp
Student stu;
stu.id = 1001;
stu.name = "小明";
stu.score = 95;
```

> 💡 竞赛中最常用方式一，简洁高效！

---

#### 2.4 结构体的空间计算（内存对齐）

C++ 中结构体存在**内存对齐**规则，实际占用内存可能比成员之和更大。

**原则：**
1. 每个成员按自身类型大小对齐（存放地址是类型大小的整数倍）
2. 结构体总大小是最大成员大小的整数倍

**示例：**

```cpp
struct A {
    char a;    // 1 字节
    int b;     // 4 字节
    double c;  // 8 字节
};

sizeof(A) = ?
```

> 分析：
> - char a：1 字节，地址 0
> - int b：4 字节，需要4字节对齐，从地址 4 开始（地址 1-3 填充）
> - double c：8 字节，需要8字节对齐，从地址 8 开始
> - 总计：16 字节（0~15），正好是 8 的倍数

```cpp
cout << sizeof(A);   // 输出 16，不是 1+4+8=13
```

**更简单的记忆方法：**

> 竞赛中不需要精确计算对齐，只需用 `sizeof` 直接查看！
>
> 关键是要知道：结构体大小 **≥** 所有成员大小之和，具体要看对齐规则。

---

#### 2.5 结构体数组

结构体和 int 一样，可以定义数组：

```cpp
Student stu[105];    // 定义 105 个 Student 结构体的数组

// 赋值
for (int i = 1; i <= n; i++) {
    cin >> stu[i].id >> stu[i].name >> stu[i].score;
}

// 遍历输出
for (int i = 1; i <= n; i++) {
    cout << stu[i].name << " " << stu[i].score << endl;
}
```

> 💡 也可以全局定义结构体数组，自动初始化为默认值（数值型为 0，string 为 ""）。

---

#### 2.6 结构体数组的 sort 排序

结合 sort 和自定义 cmp 函数，可以对结构体数组按任意规则排序：

**按成绩降序：**

```cpp
bool cmp(Student a, Student b) {
    return a.score > b.score;   // 成绩大的排前面
}

sort(stu + 1, stu + n + 1, cmp);
```

**多关键字：先成绩降序，成绩相同按学号升序：**

```cpp
bool cmp(Student a, Student b) {
    if (a.score != b.score) return a.score > b.score;
    return a.id < b.id;
}
```

**完整示例代码：**

```cpp
#include <iostream>
#include <algorithm>
#include <string>
using namespace std;

struct Student {
    int id;
    string name;
    int score;
} stu[105];

bool cmp(Student a, Student b) {
    if (a.score != b.score) return a.score > b.score;
    return a.id < b.id;
}

int main() {
    int n;
    cin >> n;
    for (int i = 1; i <= n; i++)
        cin >> stu[i].id >> stu[i].name >> stu[i].score;
    sort(stu + 1, stu + n + 1, cmp);
    for (int i = 1; i <= n; i++)
        cout << stu[i].id << " " << stu[i].name << " " << stu[i].score << endl;
    return 0;
}
```

### 3. GESP 真题演练 ⚡

**抢答题 1（选择题）：**

> 结构体定义语法正确的是（ ）
>
> A. `struct S { int a; int b; }`
> B. `struct S { int a; int b; };`
> C. `struct { int a; int b; } S;`
> D. `S struct { int a; int b; };`
>
> **答案：B**
>
> 解析：结构体定义结尾的 `}` 后必须有分号 `;`。A 缺分号；C 是另一种合法但特殊的写法（匿名结构体+变量），不是常规定义；D 语法错误。

**判断题 2：**

> 结构体变量的 `.` 运算符用于访问结构体的成员。（ ）
>
> **答案：✓（正确）**
>
> 解析：通过 `变量名.成员名` 访问结构体中的具体成员，这是 `.` 运算符的标准用法。

**抢答题 3（选择题）：**

> 以下关于结构体排序的说法，正确的是（ ）
>
> A. 结构体数组不能用 sort 排序
> B. sort 对结构体排序时必须提供自定义 cmp 函数
> C. 结构体的整体大小一定等于所有成员大小之和
> D. 同一个结构体变量可以整体赋值给另一个同类型变量
>
> **答案：D**
>
> 解析：A 错，结构体数组可以 sort（需要 cmp）；B 错，也可以重载 `operator<`；C 错，内存对齐可能使大小更大；D 正确，结构体支持整体赋值。

### 4. 进阶扩展

**在结构体定义时同时声明变量：**

```cpp
struct Point {
    int x, y;
} p1, p2, points[100];  // 定义类型的同时声明变量
```

**结构体与运算符重载（高级用法）：**

通过重载 `operator<`，可以直接 `sort` 而不需要 cmp：

```cpp
struct Student {
    string name;
    int score;
    bool operator<(const Student& other) const {
        return score > other.score;  // 降序
    }
};

sort(stu + 1, stu + n + 1);  // 使用重载的 < 运算符
```

> 黑板推演：`sort` 默认调用 `<` 来比较元素，重载 `<` 后，就 "教会" 了 sort 如何比较两个 Student 对象。`const Student& other` 中的 `const` 保证参数不被修改，`&` 是引用传递避免拷贝开销。

### 练习 5（进阶）：求结构体数组中满足条件的元素

**题目描述：**

输入 n 个学生的姓名、语文分数和英语分数，输出语文和英语都大于 80 分的学生姓名，按语文分数降序排列，语文相同按英语降序排列。

**输入格式：** n，然后 n 行姓名、语文分数、英语分数

**输出格式：** 每行一个姓名（只输出满足条件的学生）

**样例输入：**
```
4
Alice 85 90
Bob 75 88
Carol 90 82
Dave 85 95
```

**样例输出：**
```
Carol
Dave
Alice
```

**题解代码：**

```cpp
#include <iostream>
#include <algorithm>
#include <string>
using namespace std;

struct Student { string name; int chinese, english; } tmp[105], res[105];

bool cmp(Student a, Student b) {
    if (a.chinese != b.chinese) return a.chinese > b.chinese;
    return a.english > b.english;
}

int main() {
    int n; cin >> n;
    for (int i = 1; i <= n; i++) cin >> tmp[i].name >> tmp[i].chinese >> tmp[i].english;
    int cnt = 0;
    for (int i = 1; i <= n; i++)
        if (tmp[i].chinese > 80 && tmp[i].english > 80)
            res[++cnt] = tmp[i];
    sort(res + 1, res + cnt + 1, cmp);
    for (int i = 1; i <= cnt; i++) cout << res[i].name << endl;
    return 0;
}
```

#### 作业 2：成绩统计

输入 n 个学生的姓名和成绩，输出平均分和最高分学生的姓名。

**样例输入：** `3\nA 85\nB 92\nC 78`  **样例输出：** `85.00\nB`

**题解代码：**

```cpp
#include <iostream>
#include <iomanip>
#include <string>
using namespace std;
struct Student { string name; int score; } stu[105];
int main() {
    int n; cin >> n;
    double sum = 0;
    int maxScore = -1; string maxName;
    for (int i = 1; i <= n; i++) {
        cin >> stu[i].name >> stu[i].score;
        sum += stu[i].score;
        if (stu[i].score > maxScore) { maxScore = stu[i].score; maxName = stu[i].name; }
    }
    cout << fixed << setprecision(2) << sum / n << endl << maxName << endl;
    return 0;
}
```

#### 作业 3：查询结构体成员

输入 n 个矩形的宽和高，输出面积最大的矩形的宽、高和面积。

**题解代码：**

```cpp
#include <iostream>
using namespace std;
struct Rect { int w, h; } rects[105];
int main() {
    int n; cin >> n;
    for (int i = 1; i <= n; i++) cin >> rects[i].w >> rects[i].h;
    int best = 1;
    for (int i = 2; i <= n; i++)
```

初始有 n 个学生的成绩，读入 m 次更新（学号和新成绩），每次更新后输出当前成绩最高的学生。

**提示：** 每次更新后遍历找最大值（或用 sort 后取第一个）。

概念引入
03 CONCEPT

什么是结构体？

生活比喻：档案袋

每个学生有一个档案袋，里面装了：

  • 学号 (int)
  • 姓名 (string)
  • 成绩 (int)
  • 身高 (double)
结构体 = 档案袋

通过档案袋 + 隔层(.)找到具体信息

struct Student {
    int id;           // 学号
    string name;    // 姓名
    int score;      // 成绩
};

Student stu;      // 创建一个档案袋
stu.id = 1;      // 填学号
stu.name = "小明"; // 填姓名
语法详解
04 SYNTAX

结构体的定义

struct 结构体名 {
    类型1 成员名1;
    类型2 成员名2;
    // ...
};   // 注意：结尾有分号！
struct Student {
    int id;          // 学号
    string name;     // 姓名
    int score;       // 分数
    double height;   // 身高
};
⚠️ 重要提醒

结构体定义结尾的 } 后面必须有分号 ;，这是 C++ 语法规定！

变量与访问
05 USAGE

变量定义与成员访问

定义变量

Student stu1;        // 定义变量
struct Student stu2; // C风格（也支持）

访问成员（. 运算符）

stu1.id = 1001;
stu1.name = "小明";
stu1.score = 95;

cout << stu1.name << ": " 
     << stu1.score << endl;

整体赋值

Student stu2 = stu1;
// 整体复制，完全独立

stu2.name = "小红";
// 修改 stu2 不影响 stu1
💡 要点

结构体变量可以整体赋值，两个变量完全独立，互不影响。

初始化
06 INIT

结构体的初始化

方式一

按顺序（最常用）

Student stu = {
    1001,

方式二

先定义再赋值

Student stu;
stu.id = 1001;

内存对齐
07 MEMORY

结构体的空间计算

内存对齐原则

  • 每个成员按自身类型大小对齐
  • 结构体总大小是最大成员的整数倍
struct A {
    char a;     // 1字节
    int b;      // 4字节
    double c; // 8字节
};

// sizeof(A) = ?

内存布局分析

地址 0:  char a    [1字节]
地址 1-3: 填充     [3字节]
地址 4-7: int b    [4字节]
地址 8-15: double c [8字节]

总计: 16字节
cout << sizeof(A); // 输出 16
注意

不是 1+4+8=13，而是16字节（内存对齐）

结构体数组

定义与使用

for (int i = 1; i <= n; i++) {

排序
09 SORT

结构体数组的 sort 排序

自定义 cmp 函数

// 按成绩降序
bool cmp(Student a, Student b) {
    return a.score > b.score;
}

sort(stu + 1, stu + n + 1, cmp);
cmp 返回值含义

return a > b; 表示 a 应该排在 b 前面（降序）

多关键字排序

// 成绩降序，成绩相同按学号升序
bool cmp(Student a, Student b) {
    if (a.score != b.score) 
        return a.score > b.score;
    return a.id < b.id;
}
💡 记忆口诀

先比主要关键字，相等再比次要关键字

完整示例
10 EXAMPLE

完整代码示例

#include <iostream>
#include <algorithm>
#include <string>
using namespace std;

struct Student {
    int id;
    string name;
    int score;
} stu[105];

bool cmp(Student a, Student b) {
    if (a.score != b.score) return a.score > b.score;
    return a.id < b.id;
}

int main() {
    int n; cin >> n;
    for (int i = 1; i <= n; i++)
        cin >> stu[i].id >> stu[i].name >> stu[i].score;
    sort(stu + 1, stu + n + 1, cmp);
    for (int i = 1; i <= n; i++)
        cout << stu[i].id << " " << stu[i].name << " " << stu[i].score << endl;
    return 0;
}
GESP 真题
11 QUIZ

抢答题 1

结构体定义语法正确的是（ ）

A. struct S { int a; int b; }
B. struct S { int a; int b; };
C. struct { int a; int b; } S;
D. S struct { int a; int b; };
解析

答案：B。结构体定义结尾的 } 后必须有分号 ;。A 缺分号；C 是匿名结构体；D 语法错误。

GESP 真题
12 QUIZ

判断题 2

结构体变量的 . 运算符用于访问结构体的成员。（ ）

✓ 正确
✗ 错误
解析

答案：✓ 正确。通过 变量名.成员名 访问结构体中的具体成员，这是 . 运算符的标准用法。

GESP 真题
13 QUIZ

抢答题 3

以下关于结构体排序的说法，正确的是（ ）

A. 结构体数组不能用 sort 排序
B. sort 对结构体排序时必须提供自定义 cmp 函数
C. 结构体的整体大小一定等于所有成员大小之和
D. 同一个结构体变量可以整体赋值给另一个同类型变量
解析

答案：D。A 错，可以 sort；B 错，也可重载 operator<；C 错，内存对齐可能使大小更大；D 正确。

练习 2：按成绩排序

题目描述

输入 n 个学生的姓名和成绩，按成绩从高到低排序后输出。

输入：3 / Alice 85 / Bob 92 / Carol 78
输出：Bob 92 / Alice 85 / Carol 78 
#include <iostream>
#include <algorithm>
#include <string>
using namespace std;

struct Student { 
    string name; 
    int score; 
} stu[105];

bool cmp(Student a, Student b) { 
    return a.score > b.score; 
}

int main() {
    int n; cin >> n;
    for (int i = 1; i <= n; i++) 
        cin >> stu[i].name >> stu[i].score;
    sort(stu + 1, stu + n + 1, cmp);
    for (int i = 1; i <= n; i++)
        cout << stu[i].name << " " 
             << stu[i].score << endl;
    return 0;
}

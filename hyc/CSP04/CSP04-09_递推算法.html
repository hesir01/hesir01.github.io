知识回顾 — 斐波那契数列

数列：1, 1, 2, 3, 5, 8, 13, 21 ...

规律：每一项 = 前两项之和

f(n) = f(n-1) + f(n-2)

核心思想：把大问题的答案用"小一点的问题"的答案来表示

这是动态规划（DP）算法的基础！

楼梯问题引入

n 级台阶，每次走 1 步或 2 步，共几种走法？

  • 第 1 级：1 种
  • 第 2 级：2 种（1+1 或 2）
  • 第 3 级：3 种（1+1+1，1+2，2+1）
  • 第 4 级：5 种

规律：走到第 n 级 = 走到第 n-1 级 + 走到第 n-2 级

ways[n] = ways[n-1] + ways[n-2]

因为最后一步要么从 n-1 走 1 步，要么从 n-2 走 2 步

递推算法三要素
1

边界条件（初始值）

最小规模问题的答案，如 f[1] = 1

2

递推关系式

f[n] 如何由之前的 f[...] 表示

3

递推方向

从小到大（正向递推）

代码框架
// 第一步：初始化边界值 f[1] = ...; f[2] = ...; // 第二步：用循环递推 for (int i = 3; i <= n; i++) { f[i] = f[i-1] + f[i-2]; // 递推关系式 } // 第三步：输出答案 cout << f[n];

记住这个三步模板：初始化 → 循环递推 → 输出

经典问题一 — 斐波那契数列

递推关系：f[n] = f[n-1] + f[n-2]（n ≥ 3）

边界：f[1] = 1, f[2] = 1

注意：f[50] 超过 int 范围，要用 long long！

long long f[50]; f[1] = f[2] = 1; for (int i = 3; i <= n; i++) f[i] = f[i-1] + f[i-2]; cout << f[n] << endl;
经典问题二 — 楼梯走法

n 级台阶，每次走 1 步或 2 步

递推：ways[n] = ways[n-1] + ways[n-2]

边界：ways[1] = 1, ways[2] = 2

long long ways[55]; ways[1] = 1; ways[2] = 2; for (int i = 3; i <= n; i++) ways[i] = ways[i-1] + ways[i-2]; cout << ways[n];

和斐波那契数列结构相同，只是边界值不同！

经典问题三 — 杨辉三角

递推关系：C[i][j] = C[i-1][j-1] + C[i-1][j]

边界：C[i][0] = C[i][i] = 1

推导递推关系式的方法
1 列出小规模答案：手工算 n=1, 2, 3, 4, 5
2 找差值或比值规律
3 用语言描述规律
4 写成代码

例：n 个圆互相交叉，最多分成多少区域？

n=1 → 2, n=2 → 4, n=3 → 8, n=4 → 14

差值：2, 4, 6 ... → 差值是 2(n-1)

f[n] = f[n-1] + 2*(n-1)，f[1] = 2

B. 8

进阶 — 递推 vs 递归
递推 递归
方式 循环从小到大 函数自我调用
效率 高（无重复计算） 低（重复计算）
复杂度 O(n) 可能指数级

fib(40)：循环只需 40 次，递归约 2^40 次！

记忆化递归 = 递归写法 + 递推效率

课堂总结

递推核心三步：

1 初始化边界值（f[1] =, f[2] =）
2 写递推关系式（f[i] = f[i-1] + ...）
3 用循环递推到 n

推导技巧：

先算前几个值，找规律，用"上一步"描述"这一步"

递推是动态规划的基础！

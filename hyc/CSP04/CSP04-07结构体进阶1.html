新知导入

指针如何访问结构体成员？

上节课我们用 . 访问结构体成员：stu.name
但当我们使用的是结构体指针时（*p 是一个结构体），怎么访问成员？
方法一：(*p).name
先解引用，再用 .（括号必须写！）
方法二：p->name
箭头运算符，等价且更简洁 ✨
📦 就像快递：. 是直接拿货，-> 是先找到地址再拿货！
03
2.1 结构体指针

定义结构体指针

Student stu = {1001, "小明", 95};
Student* p = &stu; // p 指向 stu
  • Student* 表示指向 Student 类型的指针
  • &stu 取 stu 的地址赋给 p
  • 通过 p 可以间接操作 stu 的所有成员
// 访问成员
cout << (*p).name; // 方法一
cout << p->name; // 方法二 ✨

// 修改成员
p->score = 100;
cout << stu.score; // 输出 100
03
完整示例

结构体指针完整代码

#include <iostream>
#include <string>
using namespace std;

struct Student {
  int id; string name; int score;
};

int main() {
  Student stu = {1001, "小明", 95};
  Student* p = &stu;

  cout << p->id << " " << p->name
       << " " << p->score << endl;
  p->score = 100;
  cout << "修改后：" << stu.score;
  return 0;
}
🔑 关键总结

p->x 等价于 (*p).x

箭头运算符 -> 更常用、更易读
是"先解引用，再点运算"的简写
03
2.2 结构体嵌套

结构体里套结构体

结构体的成员可以是另一个结构体类型！
比如"学生的地址信息包含省份和城市"。
struct Address {
  string province; // 省份
  string city; // 城市
};

struct Student {
  int id; string name;
  Address addr; // 嵌套！
};
// 访问嵌套成员
stu.addr.province = "广东";
stu.addr.city = "广州";

cout << stu.addr.city;
// 输出：广州

// 初始化
Student stu = {1001, "小明",
  {"广东", "广州"}};
03
嵌套访问

用两个 . 分两层访问

Student stu;
stu.id = 1001;
stu.name = "小明";
stu.addr.province = "广东"; // 第一层 . 第二层
stu.addr.city = "广州";

cout << stu.addr.city; // 输出：广州
💡 初始化嵌套结构体有两种方式：
Address addr = {"广东", "广州"}; Student stu = {1001, "小明", addr};
或直接：Student stu2 = {1001, "小明", {"广东", "广州"}};

Part 2

🔄 函数参数传递

值传递 · 引用传递 · 指针传递

03
2.3 方式一

值传递

⚠️ 传入的是副本，修改不影响原变量
结构体较大时，复制开销大！
void printStudent(Student s) {
  cout << s.name << " "
       << s.score << endl;
  s.score = 0; // 只改副本
}
📋

特点

  • 复制整个结构体
  • 函数内修改不影响原变量
  • 简单但效率低
03
2.3 方式二

引用传递

传入的是引用，不复制，直接操作原变量！
void addScore(Student& s, int delta) {
  s.score += delta; // 直接修改原变量
}

addScore(stu, 5);
// stu.score 增加了 5
🔗

特点

  • 不复制，零开销
  • 函数内修改影响原变量
  • 需要修改时推荐使用
03
2.3 方式三

指针传递

传地址，通过 -> 访问成员
void addScore2(Student* p, int delta) {
  p->score += delta;
}

addScore2(&stu, 5);
// 传入 stu 的地址
📍

特点

  • 传地址，不复制
  • 用 -> 访问成员
  • C 语言风格，C++ 更推荐引用
03
对比总结

📊 三种传参方式对比

方式语法是否复制能否修改原变量
值传递Student s✅ 是❌ 否
引用传递Student& s❌ 否✅ 是
指针传递Student* p❌ 否✅ 是
03
2.4 const 引用

只读 + 高效

当函数只读结构体（不修改），应该用 const 引用
既避免拷贝开销，又保护数据不被修改
// const 引用：不复制，又保证只读
void printStudent(const Student& s) {
  cout << s.name << " "
       << s.score << endl;
  // s.score = 0; // ❌ 错误！
}
// sort 的 cmp 推荐
bool cmp(const Student& a,
          const Student& b) {
  return a.score > b.score;
}
🔑 最佳实践

什么时候用什么？

只读场景
const Student& s
高效且安全 ✅
需要修改
Student& s
引用传递
不关心效率
Student s
值传递，最简单

GESP 真题 · 抢答 1

选择题

设 Student* p = &stu;，以下访问 score 成员正确的是（ ）
A. p.score
B. *p.score
C. (*p).score
D. p*score
答案：C（p->score 也正确，但不在选项中）

p 是指针，不能用 .，需要先解引用 *p，再用 . 访问成员，即 (*p).score。注意括号不可省，因为 . 优先级高于 *。
⚡
GESP 真题 · 判断题 2

判断题

p->name 和 (*p).name 完全等价。（ ）
答案：✓ 正确

-> 运算符就是"先解引用，再点运算"的简写，两种写法完全等价。
⚡
GESP 真题 · 抢答 3

选择题

以下函数参数声明中，既能避免拷贝开销，又能保证函数内不修改原变量的是（ ）
A. void f(Student s)
B. void f(Student& s)
C. void f(const Student& s)
D. void f(Student* s)
答案：C

A 是值传递，有拷贝开销；B 是引用传递，可修改原变量；C 是 const 引用，无拷贝 + 只读；D 是指针传递，可通过 -> 修改原变量。
04
进阶扩展

结构体数组与指针结合

Student stu[5];
Student* p = stu + 1; // 指向 stu[1]

cout << p->name; // stu[1].name
p++; // 移动到 stu[2]
cout << p->score; // stu[2].score
🧠 黑板推演：结构体数组在内存中也是连续存储的，每个元素大小为 sizeof(Student)。
p + 1 = 地址 + sizeof(Student) 字节，这样就移动到了下一个结构体元素。
04
进阶扩展

const 指针参数

void print(const Student* p) {
  cout << p->name << " "
       << p->score << endl;
  // p->score = 0; // ❌ 错误！
}

print(&stu); // 传入地址
💡 const Student* p 表示指针指向的内容不可修改，但指针本身可以指向其他地址。

课堂练习 1（基础）

通过指针访问结构体

定义 Point 结构体（含 x, y），创建一个 Point 变量，用指针操作：输出坐标，然后将 x 和 y 各加 10，再输出。
输入：3 5 → 输出：3 5 换行 13 15
#include <iostream>
using namespace std;
struct Point { int x, y; };
int main() {
  Point pt;
  cin >> pt.x >> pt.y;
  Point* p = &pt;
  cout << p->x << " " << p->y << endl;
  p->x += 10; p->y += 10;
  cout << p->x << " " << p->y << endl;
  return 0;
}

课堂练习 2（基础）

嵌套结构体输入输出

定义 Address（省份、城市）和 Person（姓名、年龄、Address），输入一个人的信息后完整输出。
输入：小明 15 广东 广州 → 输出：小明: 15岁, 住址: 广东-广州
#include <iostream>
#include <string>
using namespace std;
struct Address { string province, city; };
struct Person { string name; int age; Address addr; };
int main() {
  Person p;
  cin >> p.name >> p.age
      >> p.addr.province >> p.addr.city;
  cout << p.name << ": " << p.age << "岁, 住址: "
      << p.addr.province << "-" << p.addr.city << endl;
  return 0;
}

课堂练习 3（综合）

用引用传参修改结构体

编写 addBonus(Student& s, int b) 和 printInfo(const Student& s)，输入 n 个学生，每个加 5 分奖励后输出。
输入：3 / Alice 85 / Bob 90 / Carol 78 → 输出：Alice 90 / Bob 95 / Carol 83
#include <iostream>
#include <string>
using namespace std;
struct Student { string name; int score; };
void addBonus(Student& s, int b) { s.score += b; }
void printInfo(const Student& s) {
  cout << s.name << " " << s.score << endl; }
Student stu[105];
int main() {
  int n; cin >> n;
  for(int i=1;i<=n;i++) cin>>stu[i].name>>stu[i].score;
  for(int i=1;i<=n;i++) addBonus(stu[i],5);
  for(int i=1;i<=n;i++) printInfo(stu[i]);
  return 0; }

void swapStudent(Student* a, Student* b) {
  Student tmp = *a; *a = *b; *b = tmp;

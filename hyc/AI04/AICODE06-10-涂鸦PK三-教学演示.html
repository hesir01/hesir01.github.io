🏗️ 第二幕：建构 Construct · 分段二

音效系统

Web Audio API：不需要任何音频文件，纯代码合成声音

// 三行代码就能发出声音！ const ctx = new AudioContext() // 1. 创建音频上下文 const osc = ctx.createOscillator() // 2. 创建振荡器（发声器） osc.connect(ctx.destination) // 3. 连接到扬声器 // 调整参数 osc.frequency.value = 300 // 频率：越高越尖锐 osc.type = 'sawtooth' // 波形：sawtooth/square/sine/triangle osc.start() // 开始发声 osc.stop(ctx.currentTime + 0.08) // 80ms后停止

常见问题 FAQ

Q1: 动画加进去后血条不显示了？
动画层可能遮挡了血条。检查 z-index 或 Canvas 绘制顺序，确保血条在最上层。
Q2: 音效每次攻击响很多次？
事件监听器被重复绑定。确保只绑定一次，或在播放前检查是否已在播放。
Q3: 死亡动画后角色还在？
动画结束回调里没有更新游戏状态。需要在动画完成后设置 isDead = true。
Q4: 特技粒子一直不消失？
粒子系统缺少生命周期管理。每个粒子需要设置存活时间，到期后移除。
Q5: 重击蓄力动画太慢？
调整蓄力阶段的持续时间。用感觉描述：「蓄力很快，0.3秒就冲出去」。
Q6: Chrome 没有声音？
Chrome 要求用户先交互才能播放音频。确保 AudioContext 在点击事件中创建。

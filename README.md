# 菠萝老师的编程课堂
> 让每个孩子都能享受编程的乐趣 🍍

## 项目简介
这是菠萝老师的少儿编程教学项目，包含**课程课件**和**学生作品**两大板块。课程涵盖图形化编程（Kitten）、AI 应用、C++ 信息学奥赛等多个方向，学生作品以 HTML 网页为主，展示每位同学的学习成果。

## 目录结构
```
├── index.html              # 菠萝老师个人主页（项目入口）
├── hyc/                    # 课程课件
│   ├── AI03/               # AI03 - 我的个人主页（上/下）交互课件
│   ├── AI04/               # AI04 - 涂鸦PK、俄罗斯方块
│   └── CSP04/              # CSP04 - C++ 结构体基础/进阶
└── student/                # 学生作品集
    ├── AICODE03/           # AICODE03 课程作品
    │   ├── huanghao/               # 黄浩
    │   ├── liwenhao/               # 李汶浩
    │   └── zhongjiaoxin/           # 钟佳鑫
    └── AICODE04/           # AICODE04 课程作品
        ├── liuzhiyuan/             # 刘致远
        ├── longziyi/               # 龙梓奕
        ├── maoweier/               # 毛威尔
        ├── wanglide/               # 王立德
        ├── xiajiangnan/            # 肖江南
        ├── xiaoxiao/               # 肖啸
        ├── yangqianli/             # 仰千里
        ├── yangshuxu/              # 杨书旭
        ├── zhouxuanyu/             # 周轩宇
        ├── zhiye/                  # 职业介绍

```

## 课程体系

| 课程 | 编号 | 适合年龄 | 内容 |
|------|------|---------|------|
| Kitten 启蒙 | — | 6-8岁 | 图形化编程，培养逻辑思维 |
| AI 应用 | AICODE03 / AICODE04 | 8-12岁 | 个人主页制作、涂鸦PK、AI工具使用 |
| C++ 进阶 | CSP04 | 9-12岁 | 信息学奥赛基础、算法思维 |
| Python 应用 | — | 10-15岁 | 数据分析、爬虫、游戏开发 |
| AI 编程 | — | 12岁+ | ChatGPT应用、AI绘画、智能项目 |

## 技术说明
- 所有课件和学生作品均为**纯前端 HTML 文件**，无需后端服务，浏览器直接打开即可
- 部分页面依赖外部 CDN（Google Fonts、Tailwind CSS、Three.js 等），需联网使用
- 学生作品中的图片资源与对应 HTML 放在同一目录下
